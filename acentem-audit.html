2.1 Kimlik Doğrulama & Yetkilendirme

Kritik~4 saatOWASP A01
Kullanıcı A, URL'deki ID'yi değiştirerek Kullanıcı B'nin poliçesine erişebilir. Her API endpoint'inde sahiplik doğrulaması şart.
@frappe.whitelist() def get_policy_detail(policy_name): # ❌ Sadece varlık kontrolü YETERSİZ # doc = frappe.get_doc("Insurance Policy", policy_name) # ✅ Yetki kontrolü if not frappe.has_permission( doctype="Insurance Policy", doc=policy_name, ptype="read" ): frappe.throw("Yetkisiz erişim", frappe.PermissionError) return frappe.get_doc("Insurance Policy", policy_name)
