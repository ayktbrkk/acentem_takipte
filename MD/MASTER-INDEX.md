## 📞 Sorun Giderme

### "Komponent bulunamadı" hatası
→ `src/components/ui/` altında ilgili komponenti kontrol et

### "CSS class tanımlı değil" hatası  
→ `src/assets/design-system.css` dosyasına ekle

### Pattern karmaşık geliyorsa
→ Benzer tamamlanmış sayfaya bak (teklifler, poliçeler)

### Stuck olduysan
→ Önceki başarılı sayfaları referans al
→ README dosyalarına bak
→ Pattern'i basitleştir
